Since I was a kid, I have loved space.The vastness of it alone amazes me - it's so big that even with all of the pencils, I couldn't begin to write down the size. The universe is full of possibilities, and although I doubt there is a planet of girl wizards, there really is no end to what could exist.

Unfortunately, being an astronaut was never going to happen. I'm scared of heights, the dark, and I have bad eyes. Luckily the internet holds the same infinite wonder. Plus I get to stay on the ground, my screen lights up, and it would almost be weird if a programmer didn't wear glasses.

I'm thrilled to be apart of something so uncharted and undefined. In 20 years, the internet will be unrecognizable to us today. I am dedicated to learning as many new languages and technologies possible so that I may have a tiny fraction of influence on what the web will become.

  • Trash Panda's Discount Critters is a fictional online shop utilizing AngularJS's UI-Routing. Each shop item is connected through the state of the application to a dynamically changing application view. Meaning that despite being a single page application, a user is able to seamlessly click through the shop items as though it was a multiple page site without suffering any delay from loading. Additionally, this site features a weather widget built using custom directives displaying the current temperature and description along with matching weather icons.
